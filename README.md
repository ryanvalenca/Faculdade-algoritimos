# Faculdade - Algoritmos

Repositório com exercícios resolvidos do livro *"Algoritmos - Lógica para Desenvolvimento de Programação"* de José Augusto Manzano, utilizados como atividade na disciplina de Algoritmos - 1º período de Ciência da Computação.

## Conteúdo

- **Cap 3 - Estruturas Sequenciais**: Entrada e saída de dados, operadores aritméticos, variáveis e tipos primitivos.
- **Cap 4 - Estruturas de Decisão**: Condicionais `if`, `else`, `else if` e `switch case`. Operadores relacionais e lógicos.
- **Cap 5 - Programação com Laços**: Estruturas de repetição `for`, `while` e `do while`. Contadores e acumuladores.
- **Cap 6 - Estruturas de Dados Homogêneas de Uma Dimensão**: Vetores/arrays unidimensionais. Declaração, inicialização e manipulação.
- **Cap 7 - Aplicações Básicas com Matrizes de Uma Dimensão**: Operações com vetores: busca, ordenação, soma, maior/menor elemento.
- **Cap 8 - Estruturas de Dados Homogêneas de Duas Dimensões**: Matrizes bidimensionais. Leitura, escrita e operações com linhas e colunas.
- **Cap 9 - Estruturas de Dados Heterogêneas**: `struct`. Agrupamento de dados de tipos diferentes em um único registro.
