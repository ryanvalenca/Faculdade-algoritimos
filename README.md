# Faculdade - Algoritmos

Repositório com exercícios resolvidos do livro *"Algoritmos - Lógica para Desenvolvimento de Programação"* de José Augusto Manzano, utilizados como atividade na disciplina de Algoritmos - 1º período de Ciência da Computação.

## 📚 Capítulos Concluídos

- **Cap. 1**: Conceitos de lógica de programação e algoritmos
- **Cap. 2**: Tipos de dados, variáveis e constantes
- **Cap. 3**: Programa com sequência - estrutura sequencial e operações de entrada/saída 
- **Cap. 4**: Estrutura sequencial e operações de entrada/saída
- **Cap. 5**: Estrutura condicional simples `if`
- **Cap. 6**: Estrutura condicional composta `if...else`
- **Cap. 7**: Estrutura de seleção múltipla `switch...case`
- **Cap. 8**: Laço de repetição com teste no início `while`
- **Cap. 9**: Laço de repetição com teste no final `do...while` e laço com variável de controle `for`
